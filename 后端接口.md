完整接口清单（按模块分类）
1. 成绩统计与可视化（6 个接口）
   GET 获取当前用户在该试卷的个人对比
   POST 导出统计数据为 Excel
   GET 导出高频错题为 Excel
   POST 获取试卷整体统计
   POST 获取可统计的试卷列表
   POST 获取用户成绩趋势
2. 错题本高级功能（19 个接口）
   POST 错题专项强化组卷（≥70% 知识点）
   POST 批量删除错题
   POST 批量分组
   POST 批量更新复习状态
   POST 批量导出错题集 (ZIP)
   POST 导出错题集为 Excel (CSV)
   GET 获取错题集导出日志
   POST 导出错题集为 PDF
   GET 导出前预览 HTML
   POST 导出错题集为 Word (.docx)
   POST 创建错题分组
   POST 删除分组
   POST 从分组中移除错题
   GET 查询分组内的错题
   GET 查询用户错题分组
   POST 添加错题备注
   GET 查询错题备注
   POST 同类错题智能推荐
   POST 推荐效果反馈
3. 错题本管理（9 个接口）
   POST 归档 - 取消归档错题
   POST 删除错题
   POST 更新错题错误类型
   GET 检查是否已分享给指定教师
   POST 分页查询错题
   POST 更新错题复习状态
   POST 分享错题集给教师（设置分享权限）
   GET 获取错题统计
   GET 获取薄弱知识点（雷达图数据）
4. 导出模板管理（5 个接口）
   POST 创建导出模板
   POST 删除导出模板
   GET 获取用户导出模板列表
   POST 设为默认模板
   POST 更新导出模板
5. 个人多维度统计（2 个接口）
   GET 获取个人多维度统计数据
   GET 导出个人统计 Excel
6. 管理员操作（4 个接口）
   POST 管理员批量删除试卷
   POST 管理员批量修改试卷状态
   POST 管理员强制删除试卷
   POST 管理员查看所有试卷（含已删除）
7. 试卷导出（7 个接口）
   POST 批量导出 ZIP
   POST 删除导出文件 (按索引)
   GET 下载导出文件
   GET 导出文件列表
   POST 导出 PDF
   POST 预览试卷 HTML
   POST 导出 Word
8. 试卷分享管理（5 个接口）
   POST 分享试卷给指定班级 - 组
   GET 获取试卷的所有分享记录
   POST 撤销试卷分享
   GET 获取分享给我的试卷
   POST 分享试卷给指定用户
9. 试卷管理（14 个接口）
   POST 添加试卷
   POST AI 智能组卷
   POST 增强版 AI 组卷（个性化提示词 + 重试）
   POST 查询 AI 组卷对话记录
   POST 从历史对话复用组卷策略
   POST 确认 AI 组卷方案并创建试卷
   POST 获取用户学习画像（AI 组卷个性化）
   POST 保存组卷结果为试卷
   POST 复制试卷（含题目关联），新试卷状态（文字截断）
   POST 逻辑删除试卷
   GET 根据 ID 获取试卷详情（含试题列表）
   POST 分页查询试卷（普通用户只能看到）（文字截断）
   POST 修改试卷状态
   POST 更新试卷
10. 试卷试题关联管理（3 个接口）
    POST 添加试题到试卷
    POST 从试卷中移除试题
    POST 更新试卷中试题的分值或排序
11. 试题管理（10 个接口）
    POST 添加试题
    POST 批量添加试题
    POST 逻辑删除试题
    GET 根据 ID 获取试题详情
    POST 分页查询试题
    POST 分页查询试题（推测为不同筛选条件的接口）
    POST 匹配计数：根据筛选条件返回各题（文字截断）
    POST ES 全文检索试题（支持分词）
    POST 修改试题状态
    POST 更新试题
12. 系统通知（4 个接口）
    GET 获取所有通知列表
    GET 获取未读通知数量
    POST 标记通知为已读
    GET 获取未读通知列表
13. 在线考试（6 个接口）
    POST 超时自动交卷
    GET 获取进行中的考试（用于恢复考试）
    GET 获取考试记录列表
    GET 查看考试成绩
    POST 开始考试（创建考试记录并返回试）（文字截断，推测为 “返回试题”）
    POST 提交考试（保存答案并自动批改客）（文字截断，推测为 “客观题”）
14. 组卷策略管理（9 个接口）
    POST 创建组卷策略
    POST 遗传算法组卷（高精度）
    POST 贪心算法组卷
    POST 复制策略
    POST 删除组卷策略
    GET 获取策略详情（含权重）
    POST 分页查询策略
    POST 设为默认策略
    POST 更新组卷策略
